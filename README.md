# APP-CONTROL-ACCESO

```bash
npm start              # local  → http://localhost:4200
npm run start:dev      # dev    → http://localhost:4200
npm run start:test     # test   → http://localhost:4200
npm run start:prod     # prod   → http://localhost:4200
```
---

## Requisitos

- Node.js >= 18
- Angular CLI: `npm install -g @angular/cli`

```bash
npm install
```
